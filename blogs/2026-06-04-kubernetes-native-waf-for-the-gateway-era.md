F5 has enhanced its web application firewall capabilities by integrating F5 WAF for NGINX with F5 NGINX Gateway Fabric alongside existing NGINX Ingress Controller support. The update addresses consistent, Kubernetes-native security enforcement as organizations transition to the Kubernetes Gateway API, introducing automated policy orchestration through Kubernetes manifests with GitOps compatibility, automated signature updates, and support for air-gapped deployments.
